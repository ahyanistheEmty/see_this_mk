
# see_this_mk

A collection of various Python utility scripts and project audit reports.

## Python Utilities
This repository contains a set of small, functional Python programs:
- `anagram_checker.py`: Checks if two strings are anagrams.
- `api_fetcher.py`: Fetches data from an API.
- `binary_to_decimal.py`: Converts binary numbers to decimal.
- `bmi_calculator.py`: Calculates Body Mass Index.
- `caesar_cipher.py`: Implements Caesar Cipher encryption/decryption.
- `calculator.py`: A basic calculator.
- `countdown_timer.py`: A simple countdown timer.
- `currency_converter.py`: Converts currency between different rates.
- `data_visualizer.py`: Visualizes data (including crypto prices).
- `dice_roller.py`: Simulates rolling a dice.
- `digital_clock.py`: A digital clock implementation.
- `word_counter.py`: Counts words in a given text.
