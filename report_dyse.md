# Repository Report: dyse

**Repository Name:** dyse
**Owner:** ahyanistheEmty
**Description:** Dyse is a clean, modern AI assistant platform with a focus on simplicity and performance, powered by Gemini 2.5 Flash. It is designed to handle tasks ranging from simple to complex, functioning as an AI agent-driven platform.
**Key Features:**
*   **AI Core:** Provides real-time voice and text interaction.
*   **Computer Use:** Allows control of a web browser via AI using Playwright.
*   **Code Forge:** Integrated code editor and visualizer.
*   **File Nexus:** Manages and analyzes local files.
*   **Cubing Hub:** Tracks and analyzes Rubik's cube solves.
*   **Graph Calc:** Offers advanced mathematical graphing.
**Technologies Used:** Node.js (v18 or later), Python (optional, for the launcher), Electron for desktop app.
**How to Run Locally:** Requires Node.js and can be launched via a Python script or npm commands.
**Deployment:** Includes instructions for building a standalone installer for Windows (.exe).
**Recent Activity:** Recent commits involve updates related to modules, UI, and general fixes.
